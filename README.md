Aplicación web interactiva diseñada como una gymkana cultural para guiar a los visitantes a través de las obras maestras del Museo Nacional Centro de Arte Reina Sofía en Madrid. El proyecto combina educación artística, gamificación y análisis de datos en una sola interfaz fluida.

## Características Principales
- Itinerario de 180 Minutos: Una ruta optimizada de 15 hitos que recorren desde la Generación de Plata hasta la arquitectura contemporánea de Jean Nouvel.
- Sistema de Progresión: Motor de juego en JavaScript que bloquea/desbloquea misiones secuencialmente para asegurar una narrativa coherente.
- Dashboard de Datos (DataViz): Visualización dinámica utilizando Chart.js para mostrar la distribución de artistas y la densidad de obras por planta.
- Línea de Tiempo Interactiva: Renderizado dinámico de una cronología artística que se activa a medida que el usuario encuentra las obras.
- Diseño Mobile-First: Optimizada con Tailwind CSS para ser utilizada cómodamente desde un smartphone mientras se recorre el museo.

## Tecnologías Utilizadas
Frontend: HTML5 semántico y CSS3.
Estilos: Tailwind CSS (vía CDN para portabilidad).
Gráficos: Chart.js para el análisis de la colección.
Tipografía: Google Fonts (Crimson Pro para textos clásicos e Inter para UI).
